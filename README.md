# Teste de conceitos - JDBC
Esse repositório tem como objetivo explorar a conectividade com bancos de dados da linguagem Java e alguns padrões de projeto presentes no ecossistema da linguagem através de um projeto de negócio simples contendo vendedores e departamentos.

A conectividade deve ser dada por um arquivo na raiz do projeto chamado ``db.properties``, instanciado como um objeto da classe Properties dentro do DriverManager. Na raiz do projeto, há um script SQL feito para simular um banco de dados real em produção.

## Ferramentas
- Java SE 21+
- MariaDB Server
- MariaDB Driver

## Conclusões
É muito esclarecedor ver como projetos mais complexos do Java são construídos olhando os fundamentos da linguagem, seu ecossistema e boas práticas. Cria uma visão mais metódica quanto ao desenvolvimento de aplicações. Tudo visa um padrão que beneficia a eficiência e mantenibilidade.
